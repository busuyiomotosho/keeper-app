# Keeper App (Inspired by Google Keep)

The Keeper App is a simple note-taking web application inspired by Google Keep. It is designed to quickly store notes/ideas locally on your computer. It's built with the ReactJS.

 ![ReactJS](https://img.shields.io/badge/-ReactJs-61DAFB?logo=react&logoColor=white&style=for-the-badge)

# Setup

## Clone the repository

```bash
git clone https://github.com/busuyi-omotosho/keeper-app.git
```

## cd into the project folder

```bash
cd \keeper-app
```

## Install the dependencies

```bash
npm install
```

## Run the app

```bash
npm start
```
